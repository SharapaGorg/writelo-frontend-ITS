# Landing Page Design: SMM Time-Saver

## Overview

Продающий лендинг для привлечения первых платящих пользователей — SMM-щиков и контент-мейкеров.

## Goals

- Привлечь первых пользователей на веб-платформу
- Конвертировать в платящих (freemium → подписка)
- Позиционирование: "ChatGPT, который уже понимает SMM"

## Target Audience

**SMM-щики и контент-мейкеры**, которые:
- Используют ChatGPT для работы
- Плохо промптят, тратят много времени
- Хотят ускорить создание контента

## Key Value Proposition

**Экономия времени** — главная ценность. Конкретные цифры: "X часов в неделю".

## Competitive Positioning

Отстройка от ChatGPT:
- Не нужно учиться промптить
- Уже настроен под SMM-задачи
- Роли и шаблоны из коробки

## Visual Style

**Linear-like минимализм:**
- Светлый фон, много воздуха
- Чистая типографика, крупные заголовки
- Один акцентный цвет для CTA
- Микро-анимации (калькулятор, числа)
- Минимум элементов, фокус на сообщении

## Page Structure

### Секция 1: Hero

**Заголовок:**
> "Контент для соцсетей за минуты, не часы"

**Подзаголовок:**
> "AI-ассистент, заточенный под SMM. Не нужно учиться промптить — выбери задачу и получи готовый результат"

**Визуал:** Скриншот/мокап интерфейса с открытым чатом и готовым постом

**CTA:** "Попробовать бесплатно"

**Микротекст:** "Без карты • 50 запросов бесплатно"

---

### Секция 2: Калькулятор времени

**Заголовок:**
> "Сколько времени ты сэкономишь?"

**Интерактивный элемент:**
- Выбор количества постов в неделю: [5] [10] [20] [30+]
- Динамический результат: "≈ 8 часов в неделю → 32 часа в месяц"
- Эмоциональный хук: "Это как дополнительный рабочий день каждую неделю"

**Логика расчёта:**
- Пост вручную: ~40 мин
- Пост с AI: ~10 мин
- Экономия: ~30 мин на пост

**Мини-CTA:** "Попробовать бесплатно →"

**Визуал:**
- Карточка с лёгкой тенью
- Анимация цифр при смене значения
- Минималистичный слайдер/кнопки

## Technical Notes

- Страница: `/landing` или отдельный поддомен
- Формат: одноэкранник (2-3 скролла максимум)
- CTA ведёт на `/auth` (регистрация)
- Калькулятор: Vue-компонент с реактивным состоянием
- Адаптив: mobile-first

## Pricing Model

- **Freemium:** 50 запросов бесплатно, 1 клиент
- **Про:** ~299₽/мес (early adopter цена, позже ~2000₽)

## Success Metrics

- Конверсия посещение → регистрация
- Конверсия регистрация → платящий
- Время на странице

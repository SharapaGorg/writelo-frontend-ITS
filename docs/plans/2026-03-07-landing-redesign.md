# Landing Page Redesign: Feature Showcase

## Overview

Редизайн лендинга в стиле Linear/Vercel с фокусом на фичи и scroll-анимации. Уход от типичного AI-стартап вида к премиум-ощущению.

## Goals

- Отстроиться от шаблонных AI-лендингов
- Показать ключевые фичи продукта
- Создать wow-эффект через scroll-анимации
- Премиум визуал в духе Linear/Vercel

## Visual Style

**Общий стиль: Linear/Vercel**
- Тёмный фон (`bg-black` / `bg-zinc-950`)
- Градиентные свечения (фиолетовый/синий, blur)
- Тонкая сетка или noise-текстура для глубины
- Крупная типографика (6xl-8xl для заголовков)
- Минимум элементов, много воздуха
- Scroll-анимации (fade-in, slide-in при появлении в viewport)

## Technical Implementation

**Библиотеки для анимаций:**
- `@vueuse/motion` или `motion-v` (Vue wrapper для Framer Motion)
- Альтернатива: CSS animations + Intersection Observer

**Компоненты:**
- `LandingHero.vue`
- `LandingFeature.vue` (переиспользуемый, с пропсами для направления анимации)
- `LandingCalculator.vue` (редизайн)
- `LandingPricing.vue`
- `LandingFooter.vue`

**Адаптив:** Mobile-first, секции stack вертикально на мобильных

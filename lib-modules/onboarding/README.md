# Онбординг

Планирует онбординг - ненавязчиво поводить пользователя по интерфейсу, чтобы показать самые основные функции.

Список остановок (очередность важна):

1) Кнопка поиска в интернете
2) Кнопка выбора роли (ОТКРЫВАЕТСЯ ПАНЕЛЬКА СО СПИСКОМ РОЛЕЙ) - просто показываем и закрываем панельку
3) Настройки - просто привести туда, не объяснять каждый пункт - снова открываем панельку, показываем и закрываем
4) Всколзь список диалогов - открываем панельку списка диалогов, но не закрываем, потому что там же будет показывать функционал проектов
5) Создание проекта, todo (не в рамках онбординга): сделать, чтобы можно было сразу вводить инструкцию при создании проекта
6) Редактирование проекта - главное хотя бы показать, что существует контекстное меню на вкладке проекта

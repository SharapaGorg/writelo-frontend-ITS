# Модуль для txt2img / img2img

Модуль размещен на отдельной странице (впервые лол)

Базовый txt2img / img2img, главная цель - привлечь сммщиков:
1) обход авторских прав с помощью img2img
2) референсы
3) иногда можно получить неплохие картинки для постов

Рефы:
- https://www.img-2-img.com/ 
- https://shedevrum.ai/ (берем возможность менять форму картинки)

Модуль для авторизации через веб (т.е. через обычный браузер, а не через телеграм миниапп)

- Форма авторизации
- Миллиард способов входа
- Апи модули
- И прочее

Todos:

1) механизм переадресации на auth (authority wall)
2) Выбрать список сервисов, апрувнуть с Алексеем
3) возможность подключить соцсети - в настройках, или создать уже отдельную страницу профиля
4) подключить к бекенду
